FLYOVER COUNTRY VISUALIZATION SUITE

The Flyover Country Visualization Suite (FCVS) is a National Science Foundation funded project designed to visualize large multivariate datasets from the Neotoma Database. The library is built on Leaflet, utilizing native icon functions and D3 visualizations. I worked on this library as a part of Flyover Country, a mobile app for geoscience outreach and data discovery.

The project is still in development, but you can interact with the example below! While temporal and taxa changes aren't yet implemented, the suite includes two types of visualizations: petal plots and bar charts.

Read more about the FCVS on Github!
